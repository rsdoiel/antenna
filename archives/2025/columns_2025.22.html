2025-05-27 Alt text on this blog

On fedi, I like alt text and I both read and write it. I hate clients that don‚Äôt display it. I want to always read it. Therefore, on my blog, the equivalent of alt text should always be readable. I could reuse the alt text as the image title attribute which is shown on mouse-over by default. I don‚Äôt like being forced to use mouse and I don‚Äôt like duplicating the description.

2025-05-31. I added a piece of Javascript to copy the alt text to the title attribute if the alt text is set and the title is not. Perhaps that‚Äôll motivate me to add more alt text.
