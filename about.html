Authors

Antenna is a proof of concept demonstrationn of a personal news aggregation. Antenna is a Link Blog. It is built using the Tools provided by Skimmer, SQLite3, Pandoc and either Bash or PowerShell.

Software Requirements

  • Skimmer >= 0.0.24
  • Pandoc >= 3.1
  • GNU Make >= 3.45
  • SQLite3 >= 3.40

Software Suggestions

  • Bash

Antenna is a personal aggregation of items found around the web. Curated with skimmer and sqlite then rendered with Pandoc.
